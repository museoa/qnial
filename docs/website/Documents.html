Documents

This page lists the documentation available from the site under the categories below: Please Click on the link to view the document or download it via ftp.

  • Q'Nial Manuals and Documentation

    • The Q'Nial for Windows Help file can be accessed independently of Q'Nial for Windows. It provides a hypertext view of the Q'Nial documentation.

    • The Q'Nial Manuals file is a zipped copy of the Q'Nial Manuals, which are in Microsoft Word format. (The manuals are somewhat obsolete and are not properly indexed.)

    • The Q'Nial Source support documents is a zipped directory of text files that includes:
      • QNial design.txt - a description the architecture of Q'Nial and the details of the source code.
      • building QNial.txt - instructions on how to build the binary versions from the source.
      • other files describing how to port the source and test its correctness.

  • Papers on Nial as a programming Language (in preparation)

  • Papers on Array Theory

    • Papers.zip is a zipped directory containing the paper "Array Theory and Nial" and a supporting font with installation instructions.
